# Git::Keys

## Installation

Add this line to your application's Gemfile:

    gem 'git-keys'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install git-keys

## Usage

    This provides an additional method to git for gathering public keys from Github.

    `git keys zph`
